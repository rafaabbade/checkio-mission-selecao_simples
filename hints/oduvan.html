Como posso acessar elementos específicos de uma tupla/lista?

Você pode acessar elementos de uma tupla/lista usando índices. Por exemplo, para acessar o primeiro, segundo e terceiro elementos de uma tupla: 

tupla = (100, 200, 300, 400, 500)
primeiro = tupla[0]
segundo = tupla[1]
terceiro = tupla[2]
Isso permitirá que você extraia os elementos desejados com base em suas posições. Lembre-se que sempre começamos a contar a partir do 0.

Como pegar o penúltimo elemento de uma lista em que não sei o tamanho?

Você pode usar índices negativos para acessar elementos a partir do final da tupla. Para pegar o último elemento, use o índice -1: 

tupla = (100, 200, 300, 400, 500)
ultimo = tupla[-1]
penultimo = tupla[-2]
Isso é útil quando você não sabe o tamanho exato da tupla e deseja acessar elementos próximos ao final.

Existe uma maneira de extrair múltiplos elementos de uma tupla de uma só vez?

Sim, você pode criar uma nova tupla contendo apenas os elementos desejados: 

tupla = (100, 200, 300, 400, 500)
nova_tupla = (tupla[0], tupla[2], tupla[-2])
Isso agrupa os elementos extraídos em uma nova tupla que pode ser usada conforme necessário.
