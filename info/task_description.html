Enunciado

Lucas, um desenvolvedor de software, precisa acessar elementos específicos em tuplas retornadas por uma API. Ele quer uma função que extraia o primeiro, o terceiro e o penúltimo elemento de cada tupla para processar apenas as informações necessárias. Sua missão aqui é criar uma função que receba uma tupla e retorne uma nova tupla com apenas 3 elementos: o primeiro, o terceiro e o penúltimo elemento da tupla fornecida.

Entrada: Uma tupla com pelo menos 3 elementos.

Saída: Uma tupla com três elementos.
